# filtro-vagas-python

Analisador de Vagas Fake (Python)

O Problema
Cansado de receber "vagas aprovadas" que na verdade são golpes para vender cursos ou cobrar taxas de material. Usei o que estou aprendendo no 1º semestre de ADS para criar um filtro de segurança.

Como funciona?
O script calcula um Score de Risco baseado em:

Pediu dinheiro? (Alerta Vermelho na hora!)

Tem site oficial?

Está na minha Lista Negra?

O que tem no código?
Funções (def) para organizar a lógica.

Loops (while) para várias consultas seguidas.

Listas para gerenciar empresas suspeitas.

Como testar (Windows)
Baixe o filtro_vagas.py.

No CMD, digite: python filtro_vagas.py.
